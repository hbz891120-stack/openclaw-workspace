3. **GitHub 同步脚本**: 系统 crontab 中有同步任务，但脚本文件需确认存在
   - 路径：`~/.openclaw/workspace/scripts/github-sync.sh`

1. **修复 Cron 错误**: 更新 EvoMap cron 任务配置，移除 WhatsApp 依赖
2. **验证 GitHub 同步**: 确认同步脚本存在并可执行
